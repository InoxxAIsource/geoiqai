## The Problem

I built MealCoreAI to 12,000 users.
ChatGPT was ranking us #1 for our category.

One morning — completely gone.
No warning. No dashboard. No alert.

Found out by manually typing my own 
product into ChatGPT.

Turns out our robots.txt was blocking 
GPTBot the entire time. A 10-minute fix.
But I had no tool to catch it.

**There is no Google Search Console for AI search.**

GeoIQ is that tool.

## CITE Framework

Every fix action is tagged with 
its GEO impact category:
[C] Citations — get cited by authoritative sources
[I] Indexability — allow AI crawlers access
[T] Trustworthiness — entity signals + schema
[E] Entity — structured data + Knowledge Graph

---

## Technical GEO Audit

GeoIQ checks 5 technical signals:

robots.txt — AI crawlers allowed?
llms.txt — does the file exist?
Schema markup — Organization JSON-LD?
Content structure — quotable content?
Entity consistency — social links present?

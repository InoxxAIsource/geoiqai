Critical UX issue found.

When a user runs an audit, fixes 
the issues GeoIQ found, then runs 
the audit again — they see the 
same old cached results.

This is confusing and breaks trust.

Fix with two changes:

CHANGE 1: Add "Run fresh audit" button

On the audit result page, 
below the score, add:

"Results cached from X hours ago"
font-size: 12px, color: #6B7280

Next to it: "Run fresh audit →" 
button (outlined, small)
font-size: 12px
border: 1px solid #4F46E5
color: #4F46E5
padding: 4px 12px
border-radius: 6px
cursor: pointer

On click:
- Anonymous users: 
  counts as one of their daily audits
  If limit reached: show upgrade prompt
- Paid users: 
  always allowed, runs immediately
- Clear cache for this domain
- Run fresh audit
- Show new results

CHANGE 2: Show cache age clearly

Current: shows results with no 
indication they are cached

New: Add banner at top of results:

If cached less than 1 hour:
  No banner — results are fresh enough

If cached 1-24 hours:
  Show subtle banner:
  bg: #FFFBEB
  border: 1px solid #FDE68A
  border-radius: 8px
  padding: 10px 16px
  margin-bottom: 16px
  
  Text: "⚡ These results are from 
  X hours ago. Fixed your issues? 
  Run a fresh audit to see 
  your updated score."
  
  "Run fresh audit →" link inline

If cached more than 24 hours 
(for paid users who get daily checks):
  Show: "Daily audit runs at 2am UTC.
  Last checked: X hours ago.
  [Run now →] to check immediately."

CHANGE 3: After paying — auto fresh audit

When a user completes Razorpay payment
and accesses dashboard for first time:
Automatically run a fresh audit 
on their domain.
Do not use cached results.
Show: "Running your first full audit..."
This gives them immediate value 
on first login.

CHANGE 4: Fix description text

Currently cache message is unclear.

Update audit result page subtitle:
If fresh: 
  "AI visibility audit · just now · 
  {category} · {market}"

If cached:
  "AI visibility audit · {X} hours ago · 
  {category} · {market} · 
  [Refresh ↻]"

The [Refresh ↻] is clickable 
and runs fresh audit.

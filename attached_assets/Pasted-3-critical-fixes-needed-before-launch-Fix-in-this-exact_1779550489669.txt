=== FIX 1: AUDIT TIMEOUT AND FAKE DOMAIN ===

Two related bugs:

Fix A — Hard timeout on domain fetcher:
Set maximum 8 seconds on all HTTP requests
during the scraping step.

In the scrape function:
const controller = new AbortController()
const timeout = setTimeout(() => 
  controller.abort(), 8000)

try {
  const response = await fetch(url, {
    signal: controller.signal,
    headers: { 
      'User-Agent': 'GeoIQ Bot' 
    }
  })
  clearTimeout(timeout)
  // process response
} catch (error) {
  clearTimeout(timeout)
  if (error.name === 'AbortError') {
    return { 
      success: false, 
      reason: 'timeout',
      domain: domain 
    }
  }
}

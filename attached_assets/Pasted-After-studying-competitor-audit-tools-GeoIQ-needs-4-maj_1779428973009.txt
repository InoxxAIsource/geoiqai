CHECK 1 — Crawler Access (robots.txt)
Fetch domain/robots.txt

- GPTBot
- ChatGPT-User  
- PerplexityBot
- Claude-Web
- anthropic-ai
- GoogleBot

All 6 allowed = 100/100 Pass (green)
3-5 allowed = 70/100 Warn (amber)
0-2 allowed = 0/100 Fail (red)

Pass: "All major AI crawlers can access 
your site"
Warn: "robots.txt found but doesn't

CHECK 2 — llms.txt File
Fetch domain/llms.txt
Check if file exists and has content

Score:
File exists with 20+ lines = 80/100 Pass

File not found = 0/100 Fail

Detail text:
Pass: "llms.txt found. Good AI accessibility"
Fail: "No llms.txt found. This file tells 
AI systems about your brand directly"

CHECK 3 — Schema Markup
Fetch homepage HTML
Check for JSON-LD script tags
Check for @type: Organization or Product

Score:
Organization schema found = 100/100 Pass
Some schema but not Organization = 50/100 Warn
No schema = 0/100 Fail

Detail text:
Pass: "Schema.org markup found. 
AI engines understand your entity"
Fail: "No schema markup found. 
AI engines struggle to identify 
your brand entity"

CHECK 4 — Content Structure
Analyze scraped homepage text
Check:
- Word count > 300 words
- Has H1, H2 headings
- Has clear value proposition
- Has factual statements AI can quote

Score:
All checks pass = 100/100 Pass
Some checks pass = 50/100 Warn
Thin content = 5/100 Fail

Detail:
Fail: "Content is extremely thin. 
AI engines have almost nothing 
to cite or quote meaningfully"

CHECK 5 — Entity Consistency
Check homepage HTML for:
- Twitter/X profile link
- LinkedIn company link
- Facebook page link
- GitHub link (for tech companies)

Score:
3+ social links = 100/100 Pass
1-2 social links = 50/100 Warn
0 social links = 0/100 Fail

Detail:
Fail: "No social profile links found. 
AI engines use social signals 
to verify entity identity"

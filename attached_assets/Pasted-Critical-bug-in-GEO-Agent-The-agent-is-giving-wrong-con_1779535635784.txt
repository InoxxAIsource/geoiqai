Pull this from the audits table:
SELECT 
  brand_name,
  category,
  market,
  raw_results->>'scraped_title' as title,
  raw_results->>'scraped_description' 
    as description,
  raw_results->>'scraped_h1' as h1,
  raw_results->>'scraped_body' as body
FROM audits
WHERE domain = $1
ORDER BY created_at DESC
LIMIT 1

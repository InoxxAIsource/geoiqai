Critical insight discovered about

=== THE REAL PROBLEM ===

ChatGPT and Gemini do NOT crawl 
websites live. They answer from 
training data with a cutoff date.

Only Perplexity uses live web search.

This means:
- New brands score 0 not because 
  their site is bad
- They score 0 because they are not 
  in AI training data yet
- The fix is getting CITED by sources
  that AI training data includes

=== FIX 1: PERPLEXITY QUERIES ===

Perplexity uses live web search.
Update Perplexity queries to be 
web-search style to get accurate results.

Current wrong approach:
Sending "What is manus.im?" to Perplexity
— this just searches for the brand name

New correct approach for Perplexity:
Send category-based discovery queries
that force Perplexity to search the web:

"best {category} tools {year}"
"top {category} software recommendations"
"{competitor1} alternatives"
"review of {brand_name}"
"is {brand_name} good for {use_case}"

Also send one direct brand query:
"What is {brand_name} {domain}?"
— Perplexity will crawl the site 
  for this one

This gives real-time web visibility data.

=== FIX 4: SMARTER CATEGORY PROMPTS ===

For Perplexity (live web search),
use these specific prompt templates
that force it to search and find 
your domain:

Template 1 — Direct site reference:
"What does {brand_name} at {domain} do?
Please visit their website and 
summarize what they offer."

Template 2 — Category search:
"Search for the best {category} tools

Template 3 — Competitor comparison:
"Compare {brand_name} with

For ChatGPT/Gemini (training data):

This forces ChatGPT to be honest 
about whether it knows the brand
rather than giving generic 
category descriptions.
